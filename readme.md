# YouTube Transcript Chatbot

## What is a YouTube Transcript Chatbot?

A **YouTube Transcript Chatbot** is an AI application that allows users to ask questions about a YouTube video.

Instead of watching the entire video, the chatbot:
- Extracts the video's transcript.
- Understands the transcript using embeddings.
- Stores the embeddings in a vector database.
- Retrieves relevant information.
- Uses an LLM to generate an answer.

It is a practical implementation of **Retrieval-Augmented Generation (RAG).**
